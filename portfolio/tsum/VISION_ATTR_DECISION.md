# TSUM — решение по атрибутам с картинок

Модель: `google/gemini-2.5-flash` · probe: **28** SKU · KEEP **89** · REJECT **5**

## Не извлекать с картинок (уже в фиде / вредно)

- **Цвет / Оттенок** — fill ~100% в фиде
- **Материал (состав %)** — fill ~91% в фиде
- **Пол** — всегда в фиде
- **Размер** — в фиде (SKU-level)
- **Бренд / vendor** — в name; топ запросов — брендовые
- **Артикул / id / штрихкод** — технические поля
- **Страна дизайна** — уже в фиде (~77%)
- **Сезон SS/FW** — уже в фиде; слабо ищут как фильтр
- **attribute_length (мини/миди/макси)** — частично заполнен (~35%); не дублировать
- **attribute_sleeve** — частично заполнен (~33%); дополнять только пробелы
- **Негации (без X)** — ломают поиск

## Извлекать с картинок (нужно партнёру)

- **Принт / узор** — в фиде нет отдельного паттерна; attribute_details ~8% и почти без клеток/полосок · запросы≈9803 / 90д
- **Силуэт / посадка** — нет structured fit; запросы оверсайз/slim/клёш · запросы≈7929 / 90д
- **Капюшон** — нет boolean hood в фиде · запросы≈1068 / 90д
- **Воротник / вырез** — нет отдельного атрибута · запросы≈n/a / 90д
- **Застёжка (визуальная)** — attribute_clasp только ~2% · запросы≈9111 / 90д
- **Детали / декор / фактура** — attribute_details sparse; стёжка, плиссе, пайетки видны на фото · запросы≈12098 / 90д
- **Тип сумки (тоут/кроссбоди/багет)** — custom categories частично; визуальный силуэт стабильнее · запросы≈18926 / 90д
- **Обувь: каблук / форма носка** — attribute_base есть, но каблук/шпилька в запросах без покрытия · запросы≈67771 / 90д
- **Парфюм: концентрация + ноты (OCR)** — в params нет нот; на флаконе часто EDP/EDT и ноты · запросы≈3385 / 90д
- **Часы: функции / циферблат / ремешок** — watch attrs sparse (<0.2%); OCR с циферблата сильный · запросы≈n/a / 90д
