ЦУМ: атрибуты с картинок — разрыв и выручка

Исследование разрыва между поисковыми запросами, текущим YML-фидом и тем, что реально видно на фото. Цель — показать, какие атрибуты уже есть и какие нужно доставать с изображений, чтобы закрывать стилевые запросы (принт, силуэт, каблук, тип сумки, декор, парфюмерные ноты).

Вывод для партнёра. Фид ЦУМа силён в «карточных» полях — бренд, цвет, материал, пол, размер. Слабое место — визуальный стиль, который покупатель формулирует в поиске и ждёт в фильтрах: принт, посадка, капюшон, застёжка, каблук, силуэт сумки, декор, OCR с флакона/циферблата. На пробе из 28 фото OpenRouter нашёл новые searchable-атрибуты у всех SKU; после фильтра коллизий с фидом — KEEP 89 / REJECT 5.
Два денежных слоя. Стрим A — сколько ₽ даёт починка уже идущих vision-запросов (RESERVE/NORMAL × CVR Метрики): база 11.9M ₽/мес. Стрим B — доп. потенциал: товары с новыми attr начинают появляться в выдаче по категориям → ещё +1.9M…+12.4M ₽/мес (cons → opt). Слои считают по-разному; полный A+B без оговорок завысит — партнёру показываем B как upside сверху базы A.

9. Методология (коротко)

  1. Инвентаризация YML (stream parse 1.5 GB) — params, fill, выборка по категориям.
  2. ClickHouse site 203 — топ-30k запросов / 90д + totals.
  3. Gap: токены запросов vs корпус фида (как lexicon/feed-collision в attributes_extraction).
  4. Vision-probe через OpenRouter (Gemini 2.5 Flash) — без локальных моделей.
  5. KEEP/REJECT: коллизия с фидом, негации, низкая search relevance.
  6. Стрим A: Diginetica API classify + Метрика CVR/AOV → Δ₽ по запросам.
  7. Стрим B: категории × P(новый attr) × доп. появление в выдаче → доп. потенциал ₽.
Что купить у нас (продуктово, без цены): пакет vision-атрибутов для поиска и фильтров — принт/узор, силуэт/посадка, капюшон, воротник/вырез, застёжка, декор/фактура, тип сумки, каблук/форма обуви, OCR парфюма и часов — с исключением дублей фида и заливкой в Dashboard feed attributes.

Ожидаемый эффект в ₽: база ~11.9M ₽/мес (стрим A) + доп. потенциал +1.9…+12.4M ₽/мес от расширения выдачи (стрим B).
