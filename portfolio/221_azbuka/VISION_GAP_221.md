## 2. Что НЕ доставать с картинок

Уже закрыто фидом / text gold / бесполезно для поиска:

| Класс | Почему skip |
|-------|-------------|
| Ноты аромата / вкус и послевкусие / характер вкуса / «ощущения» | Массово в gold; с фото модель **выдумывает**, это не OCR |
| КБЖУ, срок годности, температура хранения | Почти всегда в YML params |
| Бренд / vendor / страна / вес | Обычно в `name` + params; коллизия с фидом |
| Апелласьон / регион / сорт винограда (вино) | Уже сильные params в фиде; vision дублирует этикетку |
| ТУ / ГОСТ / «130 лет бренда» | Низкая search relevance |
| Чистый visual product_only без этикетки | Мясо на разделочной, гроздь винограда, торт на тарелке, салат/сэндвич «lifestyle» → OCR=0 |
| Цвет упаковки / декор / пропсы (розмарин, черри) | Не атрибут товара для поиска |

Правило из feed-markup: если значение уже в **названии / категории / params** — не писать атрибут и не считать «пропуском».

## 3. Что модель реально даёт с фото (probe)

Паттерн жёсткий:

- **`product_only`** (8/18 в пробе) → `new_attributes=[]`. Vision бесполезен.
- **`packshot` / `packshot_with_label`** → OCR работает; после фильтра «не дублировать name/feed/gold» остаются в основном:

### High-value (брать)

1. **Клеймы «не содержит» / диета на упаковке**  
   Примеры с чипсов Hunter's: MSG-free, No Preservatives, Non-GMO, No Artificial Flavors.  
   В фиде поле `Не содержит` почти пустое → **главный vision-uplift**.

2. **Форма выпуска / тип упаковки (когда нет в params)**  
   «мельница», стекло vs дойпак/пакет — поиск «специи мельница», «в стекле».

3. **Технология на этикетке**  
   Hand Cooked, сублимированные грибы, «в соусе / нежное филе» (pet) — если нет в name/gold.

4. **Вкус/наполнитель как LABEL** — только если написано на пачке и **нет** в названии  
   Пример: Movenpick — в имени ваниль/фундук/нуга, на пачке ещё **BROWNIE** → это gap.  
   Если «лайм и ромашка» уже в name — не дублировать.

5. **БЗМЖ / organic / безлактозный / безглютеновый** — если иконка/надпись есть, а в name/params нет.

### Low-value (не качать в прод)

- Тип упаковки «бутылка/стекло» для вина (и так очевидно + слабо ищут)
- Повтор названия линейки, если оно уже в name (Terre & Laves / Patio)
- Любые «ноты» и маркетинг без явного OCR

## 4. Фокус-категории (на них останавливаемся)

### Делаем vision (packshot-first)

| Приоритет | L1 / product_type | Зачем |
|-----------|-------------------|--------|
| P0 | **Снеки** / `snacks_and_chips` | Много англ. клеймов на пачке; text gold про «подачу», не про MSG-free |
| P0 | **Молочная продукция** (фасованная) / `milk_products` | Плотные этикетки; БЗМЖ/жирность иногда только на пачке |
| P0 | **Бакалея** — крупы/гарниры / `grains_and_legumes` + **соусы/специи** / `condiments_oils_spices_sauces` | Форма выпуска (мельница), наполнители, клеймы |
| P1 | **Вода, напитки** / `food_beverages_non_alcohol` | Packshot бутылок; тип тары + редкие клеймы |
| P1 | **Заморозка, мороженое** / `bakery_dairy` (крупная фасовка) | Линейка вкусов на крышке ≠ полный name |
| P1 | **Сухофрукты/орехи/сладкое в пачке** / `bakery_and_sweets` | Нарезка, «без сахара» и т.п. |
| P1 | **Для животных** / `pet_food_wet_dry` | Текстура/форма (филе в соусе) с этикетки |
| P2 | **Консервы/деликатесы в банке** с этикеткой | Если packshot; иначе skip |

### Не делаем vision (text + фид достаточно)

| L1 / тип | Почему |
|----------|--------|
| Овощи, фрукты, грибы (свежие) | `product_only`, атрибуты в name |
| Мясо/рыба **охлаждённые разрубы** без этикетки | Lifestyle; gold уже даёт бескостное/охлаждённый |
| Готовая еда на тарелке | Нет OCR |
| Торты/пирожные витринные | Нет OCR |
| Алкоголь / вино | Фид уже закрывает регион/сорт/апелласьон; vision ≈ дубль |

## 5. Целевая схема атрибутов для vision (нормализованные имена)

Предлагаемый closed-set для следующих прогонов (не плодить 1347 имён):

| attr | источник | пример |
|------|----------|--------|
| `Не содержит` | OCR/иконки | без ГМО; без консервантов; без глутамата |
| `Особые свойства` | OCR | organic; БЗМЖ; без лактозы; без глютена; vegan |
| `Форма выпуска` | visual+OCR | мельница; нарезка; кусочки |
| `Тип упаковки` | visual | стекло; дойпак; tetra — **только если нет в params** |
| `Технология приготовления` | OCR | hand cooked; сублимированный |
| `Вкус, Добавки` | OCR label | brownie — **если нет в name/params** |
| `Текстура корма` / `Тип соуса` | OCR (pet) | нежное филе; в соусе |

Перед записью: коллизия с name + params + gold (как «уже залито»).
